# 🎁 Secret Santa Web App (Amigo Invisible)

A web application to organize **Secret Santa / Amigo Invisible** gift exchanges with secure personal links for each participant.
Built using **Google Apps Script** and deployed as a web application.

Aplicación web para organizar sorteos de Amigo Invisible con enlaces secretos.
Desarrollada con **Google Apps Script** y desplegada como aplicación web.

---

## 🚀 Features

- Create a Secret Santa draw
- Automatic assignment generation
- Option to **show or hide assignments**
- Secure **secret links** for each participant
- **WhatsApp sharing** with prefilled messages
- **Automatic email notifications**
- Export results to **Excel**
- **Admin panel** to view all draws
- Animated reveal when opening the secret link

- Crear sorteos
- Generar asignaciones automáticamente
- Links secretos para cada participante
- Opcinal envío automático de emails
- Links directos de WhatsApp
- Exportación a Excel
- Panel admin para ver sorteos
- Animación al revelar el amigo invisible
---
